# SVG

The SVG component can be used just like the `<svg>` HTML element, but adds support for color variant on the child paths.

The SVGCircle, SVGLine, SVGPath, SVGPolygon, and SVGRect components can also be used just like their native counterparts
but accept `strokeWidth`, `stroke`, and `fill` props that support theme colors.
