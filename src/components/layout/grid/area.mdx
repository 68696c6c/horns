# Area

An Area is a generic container that supports colorways and can be assigned to a CSS grid-area.

<Playground>
  <Grid>
    <Area colorway="primary" area="left" padded>
      <Heading level="h3" align="center">Left Area</Heading>
    </Area>
    <Area colorway="secondary" area="right" padded>
      <Heading level="h3" align="center">Right Area</Heading>
    </Area>
  </Grid>
</Playground>
