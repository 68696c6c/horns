# Box

A Box is a generic, padded container that uses the CSS flex property to align its content in various ways.

<Playground>
    <Box colorway="primary" width="200px" height="200px" spacing="large">
      <Text>Large Centered Box</Text>
    </Box>
    <Box colorway="secondary" width="200px" height="200px" compact x="left">
      <Text>Left-Aligned Box</Text>
    </Box>
    <Box colorway="tertiary" width="200px" height="200px" x="right" y="bottom">
      <Text style={{ width: '50px' }}>Bottom-Right-Aligned Box</Text>
    </Box>
</Playground>
