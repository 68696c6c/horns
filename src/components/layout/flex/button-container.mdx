# ButtonContainer

A ButtonContainer is a padded container for grouping Buttons that uses the CSS flex property for alignment and supports 
Colorways.  The Buttons switch to a stacked layout at the specified breakpoint.

<Playground>
  <ButtonContainer>
    <Button colorway="primary">Left Button</Button>
    <Button colorway="secondary">Center Button</Button>
    <Button colorway="tertiary">Right Button</Button>
  </ButtonContainer>
  <ButtonContainer x="center">
    <Button colorway="primary">Left Button</Button>
    <Button colorway="secondary">Center Button</Button>
    <Button colorway="tertiary">Right Button</Button>
  </ButtonContainer>
  <ButtonContainer x="left">
    <Button colorway="primary">Left Button</Button>
    <Button colorway="secondary">Center Button</Button>
    <Button colorway="tertiary">Right Button</Button>
  </ButtonContainer>
</Playground>
