# FormGroup

FormGroup can be used to collect form inputs into a single container for styling purposes.  FormGroup are especially 
useful for collecting a set of Checkboxes or Radios under a single label.  FormGroup automatically wrap their children 
in their own containers, but do not provide any responsive behavior.

# Inline FormGroups

Inline FormGroup can be used to collect form inputs into a single row.  The inputs will be vertically aligned to each 
other and will will switch to a stacked layout at the specified breakpoint.
