# Group

Group can be used to collect form inputs into a single container for styling purposes.  Group are especially 
useful for collecting a set of Checkboxes or Radios under a single label.  Group automatically wrap their children 
in their own containers, but do not provide any responsive behavior.

# Inline Groups

Inline Group can be used to collect form inputs into a single row.  The inputs will be vertically aligned to each 
other and will will switch to a stacked layout at the specified breakpoint.
