# Header

A Header is a semantic layout element that confines its contents to the page content area and supports colorways.

  </Header>
  <Header colorway="warning">
    <p>Headers are semantic content areas that horizontally center their contents on the page.</p>
    <p>Headers also add a bit of vertical padding around their contents.</p>
    <p>Both of these features can be disabled, allowing the Header to be used as a simple themable content area.</p>
  </Header>
  <Header colorway="danger" contained={false}>

  </Header>

  </Header>
