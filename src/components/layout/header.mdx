# Header

A Header is a semantic layout element that confines its contents to the page content area and supports colorways.

<FullscreenDemo variant="dark" of={Header} title="Header">
  <Header>
    <Paragraph>These Headers have had their padding highlighted to illustrate how the padding works.</Paragraph>
  </Header>
  <Header colorway="warning">
    <Paragraph>Headers are semantic content areas that horizontally center their contents on the page.</Paragraph>
    <Paragraph>Headers also add a bit of vertical padding around their contents.</Paragraph>
    <Paragraph>Both of these features can be disabled, allowing the Header to be used as a simple themable content area.</Paragraph>
  </Header>
  <Header colorway="danger" fluid>
    <Paragraph>This Header's horizontal centering has been disabled, making it fluid.</Paragraph>
  </Header>
  <Header colorway="info" compact>

  </Header>
