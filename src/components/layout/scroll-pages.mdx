# ScrollPages
When using this component with Gatsby, you will need to use the following CSS rules.
```
  html {
    height: 100%;
  }
  body {
    height: 100%;
    margin: 0;
    padding: 0;
  }
  
  #___gatsby {
    height: 100%;
  }
  
  #___gatsby > div {
    height: 100%;
    overflow: hidden;
  }
```
