# Footer

A Footer is a semantic layout element that supports colorways.

<Playground>
  <Footer>
    <Paragraph>These Footers have had their padding highlighted to illustrate how the padding works.</Paragraph>
  </Footer>
  <Footer colorway="warning">
    <Paragraph>Footers are semantic content areas that horizontally center their contents on the page.</Paragraph>
    <Paragraph>Footers also add a bit of vertical padding around their contents.</Paragraph>
    <Paragraph>Both of these features can be disabled, allowing the Footer to be used as a simple themable content area.</Paragraph>
  </Footer>
  <Footer colorway="danger" fluid>
    <Paragraph>This Footer's horizontal centering has been disabled, making it fluid.</Paragraph>
  </Footer>
  <Footer colorway="info" compact>
    <Paragraph>This Footer's vertical padding has been disabled.</Paragraph>
  </Footer>
</Playground>
