# NavItemSticky

The NavItemSticky component is a NavItem intended for use in a sticky SiteHeader component.  The content of the 
NavItemSticky changes based on the stuck state of the SiteHeader.

<Playground>
  <NavItemSticky
    href="#"
    content={<span>Default Content</span>}
    stuckContent={<span>Stuck Content</span>}
  />
</Playground>
